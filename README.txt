PROYECTO 6

Este deber consistió en la creación y entrenamiento de un modelo que predice el valor de un número
escrito a mano. Se empleó un modelo convolucional, con y sin image augmentation. Este deber fue realizado por:

	- Carlos Montiel: imports, extracción y formateo del dataset de MNIST, y graficación sin image augmentation.
	- Juan Sánchez: creación del modelo convolucional y tests con imágenes propias.
	- Esteban Alvarado: aplicación del image augmentation y su respectiva graficación.
